# Billionaires Data Dashboard (Excel)

## Overview
This project is an interactive Excel dashboard analyzing global billionaire data.  
The dashboard provides insights into wealth distribution, industries, and geographic patterns among billionaires using data visualization techniques.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Visualization

## Dashboard Features
- Overview of global billionaire wealth
- Distribution of billionaires by country
- Industry-wise billionaire analysis
- Interactive filters using slicers
- Visual representation through charts and graphs

## Key Insights
- Certain countries dominate the global billionaire population.
- Technology and finance are among the most common industries for billionaires.
- Wealth distribution is concentrated among a few regions.
- Interactive filtering allows deeper exploration of the dataset.

## Files in this Repository
- `billionaires-dashboard.xlsx` – Excel dashboard file  
- `dataset.xlsx` – Dataset used for analysis  
- `billionaires-dashboard.png` – Screenshot of the dashboard

## Purpose
This project was created while learning Excel dashboard development and data visualization.  
It demonstrates the use of Excel for data analysis and building interactive dashboards.
